# 📊 Statistics Case Study  
A data‑driven statistical analysis project exploring real‑world datasets using Python, NumPy, Pandas, Matplotlib, Seaborn, SciPy, and Statsmodels.

## 🔍 Project Overview
This case study demonstrates practical applications of descriptive statistics, probability distributions, hypothesis testing, and regression analysis.  
It is designed as part of my ongoing learning in data analytics and statistical modeling.

##  Objectives
- Perform exploratory data analysis (EDA)
- Apply statistical tests (t‑test, ANOVA, chi‑square, correlation)
- Build and evaluate regression models
- Visualize insights using Matplotlib and Seaborn
- Interpret statistical results in a business‑relevant context
